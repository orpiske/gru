Introduction:
----

This library contains portable and reusable C code for very simple applications.

Dependencies:
----

Runtime/Compilation:
* cmake
* gcc or clang
* Microsoft Compiler (ie.: Visual Studio 2015)

Building
----

- On Linux: 
```
mkdir build && cmake ..
```

- On Windows
```
mkdir build && cmake -G "NMake Makefiles" ..
```
